# 📧 Newsletter Sign-Up Form

Este proyecto es una aplicación sencilla de suscripción a un boletín informativo.  
El usuario puede ingresar su correo electrónico en el formulario, enviarlo y recibir un mensaje de éxito personalizado.

## 🚀 Características
- Formulario de registro con validación de correo electrónico.
- Mensaje de confirmación mostrando el email ingresado.
- Estructura en **HTML**, estilos con **CSS** y validación con **JavaScript**.
- Diseño responsive y accesible.
